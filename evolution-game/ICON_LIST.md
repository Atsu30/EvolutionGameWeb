# EvolutionGame 必要なアイコン一覧（日本語名）

このゲームに必要なアイコンは以下の24種類です。
各アイコンは `public/icons/` ディレクトリに `{type}.png` というファイル名で配置する必要があります。

## アイコン一覧

1. **原始生命** (`start.png`)
   - ゲームのスタート地点。最初の生命体

2. **バクテリア** (`bacteria.png`)
   - 単純だが最強の生存者

3. **クラゲ** (`jellyfish.png`)
   - 多細胞生物、無脊椎動物の祖としても使用

4. **昆虫** (`insect.png`)
   - 節足動物、甲殻類としても使用

5. **魚類** (`fish.png`)
   - 脊椎動物の起点、シーラカンスとしても使用

6. **両生類** (`amphibian.png`)
   - 陸に上がった最初の脊椎動物

7. **爬虫類** (`reptile.png`)
   - 有羊膜類、恐竜としても使用

8. **鳥類** (`bird.png`)
   - 恐竜から進化した空を飛ぶ動物

9. **哺乳類** (`mammal.png`)
   - 単弓類、初期哺乳類、真獣類としても使用

10. **クジラ** (`whale.png`)
    - 海に戻った哺乳類

11. **人間** (`human.png`)
    - 文明を持つ霊長類

12. **イヌ** (`dog.png`)
    - 食肉目、人類の最良の友

13. **ネコ** (`cat.png`)
    - 自由気ままなハンター

14. **コウモリ** (`bat.png`)
    - 超音波を使う空飛ぶ哺乳類

15. **ウマ** (`horse.png`)
    - 草原を駆けるアスリート

16. **ワシ** (`eagle.png`)
    - 天空の王者

17. **ペンギン** (`penguin.png`)
    - 空を捨て海を選んだ鳥

18. **タコ** (`octopus.png`)
    - 頭足類、イカとしても使用

19. **カタツムリ** (`snail.png`)
    - 軟体動物としても使用

20. **ヒトデ** (`starfish.png`)
    - 棘皮動物、ウニとしても使用

21. **カモノハシ** (`platypus.png`)
    - 単孔類、哺乳類でありながら卵を産む

22. **ウーパールーパー** (`axolotl.png`)
    - 幼形成熟個体、驚異的な再生能力

23. **パンダ** (`panda.png`)
    - 肉食で竹を食べる白黒の珍獣
